# NutriType 🩸

**Nutrition Powered by Your Biology**

A personalized nutrition app based on your blood type, built with React Native (Expo).

## Features

- 🩸 Blood type-based nutrition plans (A, B, AB, O)
- 🎯 Personalized for your goal (fat loss, muscle gain, energy, etc.)
- 🥗 10 Foundation Foods for your blood type
- ⚠️ 5 Foods to Avoid
- ⚡ Metabolic Timing advice based on activity level
- 🍽️ Full sample day menu (Breakfast, Lunch, Dinner)

## Tech Stack

- React Native + Expo SDK 50
- React Navigation (Stack)
- Mock nutrition service (OpenAI integration ready)

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```

2. Start the app:
   ```sh
   npx expo start
   ```

3. Scan the QR code with **Expo Go** app on your iPhone
